Персональный журнал приступов, терапии и самочувствия

Trackly помогает наблюдать за эпилепсией спокойнее и точнее.

Приложение Trackly помогает фиксировать приступы, контролировать приём препаратов, отмечать самочувствие, хранить данные об обследованиях и формировать понятные отчёты для врача.

Открыть документы Связаться по email

Trackly не заменяет врача и не является медицинским изделием. Сервис предназначен для личного учёта и подготовки информации для медицинской консультации.

Что умеет Trackly

Сервис создаётся как личный инструмент для людей с эпилепсией, чтобы важные данные о состоянии и лечении были собраны в одном месте.

01

Журнал приступов

Фиксация даты, времени, типа приступа, возможных триггеров, тяжести и комментариев после эпизода.

02

Учёт лекарств

Расписание терапии, отметки о приёме, остаток препарата и напоминания по графику.

03

Самочувствие и триггеры

Записи о настроении, симптомах, побочных эффектах и факторах, которые могли повлиять на состояние.

04

Отчёты для врача

Экспорт сводки по приступам, лекарствам, самочувствию и обследованиям в понятном формате.

Безопасность и инфраструктура

Сайт открыто сообщает, где размещаются данные и какие внешние сервисы используются для работы Trackly.

Хранение данных

Trackly использует базу данных Supabase PostgreSQL в регионе AWS eu-west-1 (Ирландия) для хранения пользовательских данных.

Почтовая доставка

Для транзакционных и экстренных email-сообщений используется Resend. Экстренные письма отправляются с адреса alerts@mail.tracklyapp.xyz.

Домен и DNS

Домен tracklyapp.xyz обслуживается через hoster.by, а DNS-записи используются для связи домена с почтовой инфраструктурой.
