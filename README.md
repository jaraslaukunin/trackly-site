# Trackly — публичный сайт

Статический сайт проекта **Trackly** — персонального приложения для мониторинга состояния и медикаментозного лечения пациентов с эпилепсией.

Сайт содержит публичную документацию, которая используется как ссылки в приложении при регистрации и входе, а также страницу статуса сервиса.

---

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница |
| `privacy.html` | Политика конфиденциальности |
| `consent.html` | Согласие на обработку персональных данных |
| `terms.html` | Условия использования |
| `status.html` | Статус сервиса (health‑check через Supabase Edge Functions) |

---

## Технологии

- Статический HTML/CSS/JS — без фреймворков и сборщиков
- Хостинг: [GitHub Pages](https://pages.github.com)
- Бэкенд: [Supabase](https://supabase.com) (PostgreSQL, Edge Functions, Auth)
- Email-доставка: [Resend](https://resend.com)
- Домен: `tracklyapp.xyz` (DNS через hoster.by)

---

## Локализация

Сайт поддерживает четыре языка, переключаемых без перезагрузки страницы:

- **RU** — русский
- **EN** — английский
- **BY** — беларуская (афіцыйная арфаграфія)
- **BE** — łacinka (łacinizavanaja taraškievica)

Язык сохраняется в `localStorage`.

---

## Запуск локально

Никаких зависимостей и сборки не требуется. Достаточно открыть файлы напрямую или поднять любой локальный сервер, например:

```bash
npx serve .
```

или

```bash
python3 -m http.server 8080
```
